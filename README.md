# Odd Sum 

給你一個範圍 a 到 b ，請你找出 a 與 b 之間所有奇數的和。

例如：範圍 [3, 9] 中所有奇數的和就是 3 + 5 + 7 + 9 = 24 。 

輸入的第一列有一個整數 T （1≦T≦100），代表以下有多少組測試資料。

每組測試資料為兩列，包含兩個數 a 與 b （0≦a≦b≦100）。 

每組測試資料輸出一列，內容為 a 及 b 間所有奇數的和。
